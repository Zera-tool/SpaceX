# iO SpaceX

## Assumptions

- User has NodeJs installed and knows how to use NPM
- User knows how to clone a repo from github

## Install

- Open cmd prompt, locate application folder (iO-SpaceX), enter: npm i

- then: npm run

- Application is hosted on localhost:3000
